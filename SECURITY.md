# Security Policy

## Supported Versions

We aim to support the two latest minor releases of JSONMorph. Security fixes will be backported when feasible.

## Reporting a Vulnerability

If you believe you have found a security vulnerability:

1. **Do not** open a public issue.
2. Email the maintainers at `security@example.com` with a detailed description of the problem, impact, and reproduction steps.
3. Expect an initial response within five business days.

Please include any proof-of-concept code or logs needed to reproduce the issue. We appreciate responsible disclosure and will credit reporters who wish to be acknowledged.

## Preferred Languages

We can review reports submitted in English.
